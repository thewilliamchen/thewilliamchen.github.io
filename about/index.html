About Me

I'm Will.

I’m currently a software engineer at Dell EMC in the next-gen PowerStore product group, the latest midrange offering released in May 2020. I’ve been at Dell since the summer of 2017 as a summer intern, rejoining in 2018 as a full-time new graduate. I work full stack, both within my domain (Data Mobility / Data Protection), and across domains.

  • Backend (primary): PowerStore control plane; writing and unit testing Java 8/11 code, using modern technologies such as Eclipse Vert.x, ReactiveX, PostgreSQL, and Docker
    • Designing, implementing REST APIs providing local (snapshots) and remote (replication) data protection.
  • Frontend: Node.js and React; primarily in summer 2019 while the team was shorthanded
  • Functional Test: Validate product functionality, writing automated REST API tests in Python/pytest
  • Devops: Automate workflows specific to the team; improve Linux dev environment and build process
    • Jenkins, Kubernetes, Ansible, LXC/LXD containers…

Outside of work, I’m involved in middle-distance triathlons (find me on Zwift!), having been a former (sprint) swimmer in high school.

I also do side software projects, primarily distributed systems, for fun and to learn.

I love photography as well, though I haven’t been too invested in it in recent times, but hoping to make it more of a hobby in 2021 :)
